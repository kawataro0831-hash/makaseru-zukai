# 図解化スキル（MAKASERU）

AIに「〜を図解にして」と頼むだけで、見やすい図解ページ（HTML）を作ってくれる"作り方のレシピ"です。
できあがった図解は、デスクトップの「図解」フォルダに保存され、ダブルクリックでいつでも開けます。

- わからない言葉・仕組みを、絵と図で噛み砕いてくれる
- 自分の仕事の流れ（工程・段取り・役割分担）を、1枚に見える化できる
- 印刷・PDF化もワンボタン（開いた画面の右上「PDF」）

## 入れ方（Windows・3ステップ）

**① このページの緑の「Code」ボタン → 「Download ZIP」を押してダウンロード**

**② ZIPを開いて、中のフォルダを「スキル置き場」にコピー**

スキル置き場の開き方：キーボードの `Windowsキー + R` を押して、出てきた窓にこれを貼り付けてOK：

```
%USERPROFILE%\.claude\skills
```

（フォルダが無いと言われたら、エクスプローラで `C:\Users\自分の名前` を開き、`.claude` フォルダ →その中に `skills` フォルダを作ってください）

コピーしたら、フォルダ名を `zukai` に変えておくとわかりやすいです。

**③ Claudeアプリを一度終了して、開き直す**

これで準備完了。入力欄で `/` を打つと「zukai」が出てきます。

## 使い方

チャットにこう書くだけです：

```
受注から納品までの流れを図解にして
```

```
APIについて調べて、分かりやすくまとめて
```

わからない言葉を頼んだ時は、AIがまず**ネットで網羅的に調べて**、専門用語を新入社員にも分かる言葉とたとえ話に**噛み砕いてから**、図解にしてくれます。

できあがると、デスクトップの「図解」フォルダにHTMLファイルが保存され、ブラウザで自動で開きます。

## URLで共有する（希望者向け）

できた図解は、頼めば**URLにして共有**できます：

```
この図解、URLにして共有して
```

AIが「Surge」という無料サービスにアップロードして、`https://zukai-〇〇.surge.sh` のようなリンクを作ってくれます。リンクを送るだけで、相手はスマホからでも見られます。

- 初回だけ、AIの案内にそって無料アカウントを作ります（メールアドレスとパスワードは**自分で**入力します）
- **URLを知っている人は誰でも見られます**。社外秘の内容（取引先名・金額など）が入った図解は共有しないでください
- アップロードができない環境の場合は、デスクトップの「図解」フォルダのHTMLファイルをメールやチャットに**添付**すればOKです（相手はダブルクリックで開けます）

## お手本

`references/model-answer.html` をブラウザにドラッグ＆ドロップすると、完成品のイメージが見られます（「受注から納品までの流れ」の図解です）。

## 困ったら

| 症状 | 対処 |
|---|---|
| `/` を打っても zukai が出ない | 置き場所（上の②）を確認 → Claudeを完全に終了して開き直す |
| 図解がブラウザで開かない | デスクトップの「図解」フォルダの中のファイルをダブルクリック |
| デザインが崩れて見える | インターネット接続を確認（デザイン部品をネットから読み込むため） |

それでも解決しなければ、MAKASERU（川口）まで。
